## Microsoft Office

微软公司开发的一款办公软件套件。它包括了常用的文字处理软件 Microsoft Word、电子表格软件 Microsoft Excel、演示文稿软件 Microsoft PowerPoint 以及邮件客户端软件 Microsoft Outlook 等多个工具。Microsoft Office 使得用户可以更加高效地完成文档编辑、数据分析、信息共享、邮件管理等各种日常办公任务。该软件在全球范围内广泛应用于家庭、企业和学校等场景

- 推荐安装方式，第三方工具：Office Tool Plus
- 第三方传送门：[https://otp.landian.vip](https://otp.landian.vip)

## Adobe

美国跨国软件公司 Adobe 以其创造卓越的数字体验而闻名，其产品包括 Photoshop、Illustrator、InDesign、Acrobat、Dreamweaver、Premiere Pro、After Effects、Lightroom 等。Adobe 于 1982 年 12 月由 John Warnock 和 Charles Geschke 创立，是世界上最大的软件公司之一

- 推荐安装方式，微博搜索：嬴政天下 vposy

## Google Chrome

由 Google 开发的免费网页浏览器，具有快速、安全、简洁等特点，支持 Windows、macOS、Linux、Android 和 iOS 等多个平台。它基于 Blink 渲染引擎，拥有丰富的扩展生态系统，可以满足几乎所有的上网需求。Chrome 有书签管理、隐私模式、广告拦截、多标签页、多账号登录等功能，是目前使用最广泛的浏览器

- 传送门：[https://www.google.com/chrome](https://www.google.com/chrome)

## IDEA Community

由 JetBrains 开发的 Java 集成开发环境，是 IntelliJ IDEA 的社区版本。支持 Java、Kotlin、Groovy、Scala、Android、Maven、Gradle 等多种编程语言和框架。IDEA Community 具有代码自动补全、代码检查、重构、版本控制等功能，可以帮助开发者提高编码效率

- 传送门：[https://www.jetbrains.com/idea](https://www.jetbrains.com/idea)

## Visual Studio Code

由微软开发的免费、开源的代码编辑器，支持 Windows、macOS 和 Linux 等多个平台。它内置了 JavaScript、TypeScript 和 Node.js 支持，拥有丰富的扩展生态系统，可以满足几乎所有的开发需求。VS Code 具有智能代码补全、代码片段、版本控制、代码重构等功能，可以帮助开发者提高编码效率

- 传送门：[https://code.visualstudio.com](https://code.visualstudio.com)

## Alacritty

开源的跨平台终端模拟器，使用 Rust 编写，基于 GPU 加速，具有高性能和低资源占用的特点

- 传送门：[https://github.com/alacritty/alacritty](https://github.com/alacritty/alacritty)
- 主题推荐：[https://github.com/alacritty/alacritty-theme/blob/master/themes/material_theme.yaml](https://github.com/alacritty/alacritty-theme/blob/master/themes/material_theme.yaml)

## Git

开源的分布式版本控制系统，可以用于管理软件开发过程中的源代码。Git 以其高效、灵活、易用等特性而闻名，是目前最流行的版本控制系统

- 传送门：[https://git-scm.com](https://git-scm.com)

## MinGW-w64

- 传送门：[https://sourceforge.net/projects/mingw-w64/files/mingw-w64/mingw-w64-release](https://sourceforge.net/projects/mingw-w64/files/mingw-w64/mingw-w64-release)

## OpenJDK

开源的 Java 开发工具包，包含了 Java 编译器、Java 虚拟机、Java 类库等

- 传送门：[https://openjdk.org](https://openjdk.org)

## adb/fastboot

ADB 是一个命令行工具，用于与运行 Android 操作系统的设备进行通信。通过 ADB，开发人员可以在计算机上使用命令来管理和调试 Android 设备，例如安装应用程序、复制文件、发送 shell 命令等。ADB 还提供了许多有用的功能，如日志记录、端口转发、屏幕截图

- 传送门：[https://developer.android.com/studio/releases/platform-tools](https://developer.android.com/studio/releases/platform-tools)

## foobar2000

免费的音频播放器，提供了一个简单而直观的用户界面，支持多种音频格式，具有广泛的插件支持，允许用户根据自己的需要和喜好进行定制。用户可以添加可视化效果、支持不同的 DSP 插件、更改外观和布局等等。此外，foobar2000 还提供了许多高级功能，例如重播增益、音频标签编辑、转换、CD提取等。是一款非常受欢迎的音频播放器，尤其受到那些注重音频质量和灵活性的用户的喜爱

- 传送门：[https://www.foobar2000.org](https://www.foobar2000.org)
- 第三方美化包传送门：[https://github.com/dream7180/foobox-cn](https://github.com/dream7180/foobox-cn)

## MusicTag

免费的音乐标签编辑器，可以帮助用户编辑音乐文件的元数据。支持批量编辑音乐文件的元数据，包括歌曲名、歌手、专辑、封面、歌词等

- 传送门：[https://www.cnblogs.com/vinlxc/p/11347744.html](https://www.cnblogs.com/vinlxc/p/11347744.html)

## K-lite Codec Pack

一款免费的音视频解码器套件，支持 Windows 操作系统。它包含了多种音视频解码器，可以帮助用户播放几乎所有的音视频格式

- 传送门：[https://codecguide.com](https://codecguide.com)

## ImageGlass

开源的 Windows 图像查看器，支持多种图像格式，旨在提供一个快速、简单和易于使用的图像浏览器。同时也具有一些高级特性，提供了多个视图模式、缩略图预览、幻灯片放映、截屏、翻转、调整大小等功能以及扩展插件

- 传送门：[https://github.com/d2phap/ImageGlass](https://github.com/d2phap/ImageGlass)

## mpv

开源的视频播放器，可以播放本地视频文件和在线视频。支持 MKV、MP4、AVI、MOV、WMV、FLV 等多种视频格式，H.264、H.265、VP9、AV1 等多种视频编码格式，MP3、AAC、FLAC、WAV、APE 等多种音频格式，SRT、ASS、SSA、VTT 等多种字幕格式，UTF-8、UTF-16、UTF-32、GBK、BIG5 等多种字幕编码格式

- 传送门：[https://mpv.io](https://mpv.io)

## OBS Studio

免费、开源的跨平台电脑直播和录屏软件。可以用于直播游戏、教育、演讲、活动等多种场景，同时也支持录制桌面、摄像头和麦克风等音视频设备。用户可以自定义窗口大小、分辨率、比特率等参数，并支持输出到多个流媒体平台

- 传送门：[https://github.com/obsproject/obs-studio](https://github.com/obsproject/obs-studio)

## ffmpeg

开源的音视频处理工具，可以用于对音视频进行解码、编码、转码、混流、分离、合并、剪切、滤镜等操作。支持多种音视频格式，可以用于音视频处理、转换和编辑

- 第三方传送门：[https://github.com/BtbN/FFmpeg-Builds](https://github.com/BtbN/FFmpeg-Builds)

## Sigil

开源的跨平台 ePub 电子书编辑器，它提供了一种方便的方法来创建、编辑和修复 ePub 格式的电子书。支持多种操作系统，如Windows、macOS 和 Linux 等，提供了 WYSIWYG 编辑器和 Code View 视图两种编辑方式，用户可以轻松地添加、修改和删除文本、图片、超链接等内容。此外，Sigil 还支持文本搜索和替换、元数据编辑、代码检查等功能，以及多种 ePub 验证工具

- 传送门：[https://github.com/Sigil-Ebook/Sigil](https://github.com/Sigil-Ebook/Sigil)

## 7-Zip

## Steam

数字游戏发行平台，由美国公司 Valve Corporation 开发和运营。用户可以在 Steam 上购买、下载和游玩游戏，也可以与其他玩家交流或参加各种游戏活动。Steam 于 2003 年正式上线，最初只提供 Valve 公司自己的游戏，如 Half-Life 系列、Counter-Strike 系列等。随着时间的推移，越来越多的游戏开发商和发行商加入 Steam 平台，使其逐渐成为PC游戏发行的主要平台之一。除了游戏销售之外，Steam 还提供了诸如游戏社区、在线对战、游戏免费试玩等功能。此外，Steam 还允许用户制作和发布自己的游戏模组，使玩家可以在原有游戏基础上进行创作和分享。目前已经成为全球最大的数字游戏发行平台

- 传送门：[https://store.steampowered.com](https://store.steampowered.com)

## WeGame

- 传送门：[https://www.wegame.com.cn](https://www.wegame.com.cn)

## Epic Games
