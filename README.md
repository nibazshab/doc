# README

[![](https://img.shields.io/badge/nibazshab-wiki-9cf?style=flat-square&logo=appveyor)](https://github.com/nibazshab/wiki)

这是一个使用 VuePress 2 搭建的 Wiki，用于记录个人的学习笔记

本仓库包含以下内容：

1. `.github/workflows` 用于 GitHub Actions 的配置文件
2. `docs` 存放 VuePress 2 资源的目录
3. `README.md` 本仓库的说明文件
4. `LICENSE` 本仓库使用的许可证
5. `package.json` 生产 VuePress 2 所需依赖的配置文件
6. `.gitignore` 用于 Git 忽略的文件
