# README

这是一个使用 VuePress 2 搭建的文档站点，用于记录个人的学习笔记

本仓库包含以下分支：

- `main` 仓库的主分支，用于存放文档等原始文件
- `gh-pages` 用于存放 VuePress 2 生成的静态网页，使用 GitHub Actions 自动部署  

本仓库分支包含以下内容：

- `.github/workflows` 用于 GitHub Actions 的配置文件
- `docs` 存放 VuePress 2 资源的目录
- `.gitignore` 用于 Git 忽略的文件
- `LICENSE` 本仓库使用的许可证
- `README.md` 本仓库的说明文件
- `package.json` 描述 VuePress 2 项目的配置文件
