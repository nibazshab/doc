Stack - Hugo主题

# 信息

Github：hugo-theme-stack (opens new window)

# 删除页脚 powered by 信息

打开 hugo-theme-stack/i18n/，删除对应的 .yaml 中的以下内容

footer:
    builtWith:
        other: Creado con {{ .Generator }}

    designedBy:
        other: Tema {{ .Theme }} diseñado por {{ .DesignedBy }}

# 修改英文字体

在 hugo-theme-stack/layouts/partials/head/custom.html 中写入

<style>
    :root {
	--base-font-family: Menlo, Monaco, Consolas, "Courier New";
    }
</style>
